IF SOMETHING GOES WRONG
================================================================================

Most Common Issues:

❌ Build fails
   └─ Check Vercel logs for error messages
   └─ Usually missing environment variables

❌ 500 errors on API calls
   └─ Verify DATABASE_URL and JWT_SECRET in Vercel
   └─ Settings → Environment Variables

❌ Still getting FUNCTION_INVOCATION_FAILED
   └─ Clear cache: Vercel Dashboard → Deployments → Redeploy
   └─ Force refresh: Ctrl+Shift+R (hard refresh)

❌ Database connection error
   └─ Test DATABASE_URL locally first
   └─ Verify it's set in Vercel environment variables
