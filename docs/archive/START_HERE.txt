THREE-LAYER ERROR PROTECTION
────────────────────────────────────────────────────────────────────────────────

Layer 1: asyncHandler Wrapper
  └─ Catches async errors in route handlers
  └─ Passes to Express error middleware

Layer 2: Error Middleware (errorHandler)
  └─ Formats errors as JSON
  └─ Sets proper HTTP status codes
  └─ Returns structured response

Layer 3: Process Listeners
  └─ Catches any unhandled rejections
  └─ Logs critical errors
  └─ Prevents total function crash
